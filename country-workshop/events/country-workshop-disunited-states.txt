# Most differences from the original Disunited States code are highlighted by comments. Extra
# couple things:
#
# - Some pop tweaks are required for this start to work:
#   * all the breakaway countries are granted the same starting `america_new_world` modifier as the
#     one the United States starts with (this includes Hawaii, Alaska, Texas)
#   * European immigration (in the form of soldier pops) is fast-forwarded for the Western and
#     Miswestern breakaway countries so that they can each sustain a small deterrence force
#   * some select political reforms are enabled for the Western and Midwestern breakaway countries
#     to be able to compete for immigration with the East Coast and Texas
#
# - To compensate for their low overall ranks and vulnerable starting positions, there are a couple
#   military adjustments targeting the smaller (esp. Western) countries:
#   * military stockpile tweaks, granting up to ~10 infantry brigades worth of supplies for 10 years
#     of war to the most needy countries
#   * add a couple brigades, which is tricky but together with the pop tweaks & a save/reload should
#     do the job
#
# - The FSA is kept in check in a couple places, as it can easily turn into a powerhouse even in the
#   inept hands of the AI.
#
# - The other larger countries (FSA, CSA, NEN) are still given a small stockpile to be able to build
#   4 brigades worth of infantry. This compensates for the fact that the transfer of Union troops is
#   seemingly unreliable.
#
# - Grant 20k extra cash to every country. Again, to compensate for having to rebuild with low rank.
#
# Implementation notes:
# - The order of executing events is significant, refer to the enabling event.
# - The soldier pop tweaks have a story of their own:
#   * the original non-accepted soldier pops in the destination province(s) are moved to a sensible
#     province (i.e. nearby, preferably in the same state) with a matching pop for merging purpose
#   * an accepted soldier pop of an appropriate religion is either left, or moved into the
#     destination province(s)
#   * European soldiers of the appropriate religion are picked at random and moved to the
#     destination, and merge into the intended soldier pop or pops
#   * we forbid bringing soldiers from locations which start with large soldier pops:
#     - London (300)
#   * some destinations such as Chicago (New American Alliance) get to have one pop of each religion
#   * merging can't be performed in Salt Lake City (Deseret) due to the intended soldiers having the
#     Mormon faith
#   * moving Protestant soldiers seems to be very, very tricky! if the originating province has both
#     a Catholic and a Protestant soldier pop, it appears that the game occasionally selects the
#     Catholic pop *even* when the code calls for specifically selecting Protestant soldiers (this
#     doesn't seem to be an issue the other way around)
#   * we circumvent the above issue by only ever picking Protestant soldiers from provinces without
#     Catholic soldiers present
#   * this usually means that Protestant moves are smaller per individual pop moved
# - `random_country` is used in select spots as some effects don't come through to the intended
#   country otherwise (is this a `change_tag`-style scope invalidation issue? is this because
#   `immediate` code is extra fussy? who knows). It's also used in other spots even when not
#   required, for the sake of consistency.
# - The duration of "No More War" is not really expected to be random, due to game limitations.
